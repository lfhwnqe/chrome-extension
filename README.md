# chrome-extension

> chrome 插件开发文档，包含三个分支 `feature-qrcode`、`feature-ad`、`feature-buy`。

- `feature-qrcode` 对应二维码插件代码
- `feature-ad` 对应去除广告插件代码
- `feature-buy` 对应购物助手代码
